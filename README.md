# AI 정의: 
* AI는 기계가 사람과 유사하게 과제를 수행하는 기술로 딥러닝과 자연어 처리기술을 통해 대량의 데이터를 처리하고 패턴을 인식하는 것

## AI 세그먼트
* 머신 러닝(Machine learning)
* 자연 언어 처리 및 이해 (Natural language processing and understanding)
* 컴퓨터 비전(Computer vision)
* 기계 추론(Machine reasoning)
* 강력한 AI(Strong AI)
* 딥 러닝(Deep learning)
